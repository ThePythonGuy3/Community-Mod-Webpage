Mindustry Community Mod


About


An unofficial mindustry mod maintained and driven by the mindustry modder community. Involving their creativity and intelligence creating contents based on the theme set by the community itself. Themes are going to be changed every month.


Participating


You can participate by creating a pull request with creative contents according to the current theme.

Theme


The current theme for the mod is Mechanical Madness. participate now before the current theme is over! List of themes is available in the theme log.
